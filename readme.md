## 📦 Description

A fast [Interval Tree](https://en.wikipedia.org/wiki/Interval_tree) data structure for python:

- [x] **Typed** Uses generics in intervals and tree classes
- [x] **Fast**: Binned sorage reduces problem complexity

## 📦 Usage

Simply add the [`quicktree`](https://pypi.org/project/quicktree/) PyPI package to your `pyproject.toml` and use it:

```toml
[project]
dependencies = ["quicktree"]
```
